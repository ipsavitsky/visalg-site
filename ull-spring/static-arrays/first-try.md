# Условие
Попробуем использовать массивы. Задание простое: завести массив на 6 элементов целочисленного типа. Считать все 5 элементов и вывести их в обратном порядке. 


# Ввод
```
5
3
2
1
7
5
```

# Вывод
```
5 7 1 2 3 5
```
