# Условие
Написать функцию `Hypotenuse`, которая будет принимать в себя длину двух катетов и возвращает длину гипотенузы.

Ввести два числа с клавиатуры, с помощью функции рассчитать длину гипотенузы и вывести ее

# Ввод
```
3 4
```

# Вывод
```
5
```
